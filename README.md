# JD Pressure Washing

A work-in-progress demo site built for a potential prospect being landed through Spark Automation, our AI automation agency for local service businesses.

## How this was built

Sparsh picked the prospect and made the calls that mattered: which services to lead with, using a real Yelp review instead of only placeholders, and pushing for a before/after slider so a skeptical business owner could see the work instead of just reading about it.

This was built in close collaboration with Claude, and it went through two real passes — a base version, then a full upgrade (gallery, testimonials, sticky mobile CTA) — because the first draft wasn't good enough to send. Pricing was adjusted after review too. Both versions were checked live on Vercel before being texted to the actual prospect.
