# JD Pressure Washing

A work-in-progress demo site built for a potential prospect being landed through Spark Automation, our AI automation agency for local service businesses.

## How this was built

Built through a genuine back-and-forth with Claude — ideas for direction, content, and features were proposed, tested, and refined together, not accepted on a first draft. The first version wasn't good enough and got a full second pass — gallery, testimonials, sticky mobile CTA — with pricing and content adjusted along the way. Both versions were checked live before being sent anywhere near the actual prospect.
